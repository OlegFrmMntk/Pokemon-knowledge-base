Fairy type

The fairy type is powerful before the dragon. Dragon type attacks have no effect on the fairy. Also, the fairy type deals double damage to combat and dark type Pokémon. Half damage to fairy type Pokémon is done by fighting, insect and dark type Pokémon. fairy Pokémon are weak to poison and steel attacks. Half-damage fairy type Pokémon inflict poisonous, steel, or fire type Pokémon.
